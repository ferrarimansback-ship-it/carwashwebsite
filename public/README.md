# Spotless Sams

## Setup Instructions
1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd spotless-sams`
3. Install dependencies: `npm install`
4. Configure Tailwind CSS by adding the necessary imports in `globals.css`.

## How to Run Locally
1. Start the development server: `npm run dev`
2. Open your browser and navigate to `http://localhost:3000` to view the application.

## How to Export as Static Site for GitHub Pages
1. Build the application for production: `npm run build`
2. Export the static site: `npm run export`
3. Deploy the contents of the `out` directory to GitHub Pages.
